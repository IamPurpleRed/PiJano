## Project Features
- **讀寫檔**: 把節奏譜寫成txt檔，音樂轉成wav檔，遊戲開啟時會讀取檔案，只要有同名的兩個檔案，就會顯示在選擇介面上，因此玩家也可以自創節奏譜和音樂放進專案
- **網路功能**: 多人遊玩時可以及時顯示分數及排名
- **搖桿操控**: 遊戲可以用搖桿操控
- **遊戲玩法**: 介面仿照鋼琴塊，分成**四個區域**對應搖桿的 左鍵, 右鍵, 正方形鍵, 圓形鍵，如果是**連打(滑行)** 則需要利用 R1 或 R2 搭配蘑菇頭
